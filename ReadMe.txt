Aqui se representa trafico de automoviles. Representamos con:
0 = Celda Vacia
1 = Auto que va hacia la Derecha
2 = Auto que va hacia Arriba

Los autos que se desplazan hacia arriba tienen prioridad. Es decir, si se
presenta la situacion:
                            1 0     Donde tanto 1 como 2 pueden moverse a
                              2     la casilla donde esta el 0, quedara:

                            1 2
                              0
